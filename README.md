## 小白教程

0. 安装[Node.js](https://nodejs.org/en/)

1. 下载代码，vscode打开根目录，终端执行

   ```
   npm install   //下载项目依赖
   ```

2. 本地调试（退出 Ctrl + C）

   ```
   npm start
   ```

3. 打包


   终端运行

   ```
   npm run build
   ```

4. 部署

   打开生成的dist目录，复制里面的内容到你的网站上即可
